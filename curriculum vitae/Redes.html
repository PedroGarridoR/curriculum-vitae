¿EN QUE CONSISTE ESTA ASIGNATURA?

Tratamos las bases de las redes, empezando por los elementos que la forma, como los host, y como estos, mediante procesos, se relacionan entre sí

Hasta ahora hemos aprendido direccionamiento básico IP, y como los host necesitan saber las direcciones MAC de los demás host de la red, para que puedan comunicarse entre sí, guardando dicha dirección en la tabla CAM.

Para comprender el proceso que realizan los host para comunicarse entre sí, hemos usado herramientas como WireShark para observar los paquetes que se envían los host entre sí y el contenido de estos, y de esta forma, interpretarlos.
